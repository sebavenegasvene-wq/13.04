# Repositorio 01 HTML CSS GIT

## Nombre: Luis Arriagada

## Paso 0 (si no lo tiene)

1. Descargar e instalar GIT.
2. Descargar e instalar VSC.
3. Crear un directorio para trabajar y abrirlo con VSC.
4. Abrir la terminal y ejecutar:

```bash
git config --global user.name "nombre_usuario"  # agregar nombre de usuario
git config --global user.email "correo"         # agregar correo del usuario
```

## Paso 1

1. Dentro de un directorio, clonar el proyecto:

```bash
git clone http:\\....git
```

2. Abrir la carpeta clonada con VSC.
3. Para guardar los cambios:

```bash
git add .                          # prepara los archivos (staging)
git commit -m "comentario"        # crea un punto de guardado
git push origin main               # sube los archivos a GitHub
```
